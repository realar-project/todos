[Realar](https://github.com/betula/realar) TodoMVC implementation.

Try It on your computer.

```bash
git clone git@github.com:realar-project/todos.git
cd todos
npm i
npm run start
# Open http://localhost:1234 in your browser
```

Enjoy!
